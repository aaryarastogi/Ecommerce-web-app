It is a platform where users login and can purchase any item by their choice.

**Features**
1. User authentication
2. Adding item to cart.
3. Purchasing item by doing payment.

**Technologies:**
MERN stack, Material UI etc.

To run this project:
1. CLone this project using command: git clone git@github.com:aaryarastogi/Ecommerce-web-app.git
2. Go to that folder.
3. Go to the server folder 
4. Run the server using command : npm start
5. Go to the client folder.
6. Run the client using command: npm start
7. View it on url: http://localhost:3000
